# 如何把字典编码转为值？

private static final String USER_CODE = "$user";//用户
private static final String ORG_CODE = "$org";//机构
private static final String REGION_CODE = "$region";//区划

那字典怎么转码呢?

直接美元符加上字典类型$dicType 就可以啦
