# 搭建方法

  • 代码拉取：

首先，找到管理员授权 git 工程 A 权限。

  • 拉取工程：

找到本地盘目录，搜索框输入 cmd 打开 DOS 命令行或右键点击 Git Bash Here

输入 git clone + （工程 A 的 git 地址）

  • 使用 IDEA 打开工程,依次点击上边的 File > Open...
